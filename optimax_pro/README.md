# Optimax Pro

![Optimax Pro Logo](assets/logo.png)

## Современная утилита для оптимизации Windows

Optimax Pro - это мощная утилита для оптимизации, очистки и обеспечения безопасности операционной системы Windows. С помощью Optimax Pro вы можете ускорить работу компьютера, освободить место на диске и защитить свою систему от угроз безопасности.

### Основные возможности

- **Мониторинг ресурсов**: Отслеживание использования CPU, RAM и диска в реальном времени
- **Оптимизация системы**: Более 50 различных твиков для увеличения производительности
- **Очистка системы**: Удаление временных файлов, кэша браузеров и других ненужных данных
- **Повышение безопасности**: Настройка брандмауэра, политики паролей и других параметров безопасности
- **Управление автозагрузкой**: Контроль программ, запускающихся при старте системы
- **AI анализ**: Интеллектуальный анализ системы и рекомендации по оптимизации
- **Профили оптимизации**: Предустановленные профили для разных сценариев использования
- **Планировщик задач**: Автоматизация задач оптимизации и очистки

### Требования к системе

- Windows 10/11 (64-bit)
- 4 GB RAM или более
- 100 MB свободного места на диске
- Права администратора для применения твиков
- Python 3.9 или выше (если запускается из исходного кода)

### Установка

#### Запуск из исходного кода

1. Установите Python 3.9 или выше
2. Клонируйте репозиторий: `git clone https://github.com/username/optimax-pro.git`
3. Перейдите в папку проекта: `cd optimax-pro`
4. Установите зависимости: `pip install -r requirements.txt`
5. Запустите приложение: `python main.py`

#### Запуск из архива

1. Распакуйте архив в любую папку
2. Запустите `optimax_pro.exe`

### Использование

1. **Мониторинг**: Отслеживайте использование ресурсов в реальном времени
2. **Оптимизация**: Выберите нужные твики и нажмите "Применить"
3. **Очистка**: Освободите место на диске, удалив ненужные файлы
4. **Безопасность**: Настройте параметры безопасности системы
5. **Профили**: Используйте готовые профили оптимизации для разных сценариев

### Безопасность

Optimax Pro использует только безопасные методы оптимизации и очистки. Однако, некоторые твики могут влиять на работу системы, поэтому рекомендуется создавать точку восстановления перед применением большого количества твиков.
